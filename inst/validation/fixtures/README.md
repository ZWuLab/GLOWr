# Test Data Fixtures

This directory contains pre-generated test datasets for GLOWr validation.

All fixtures are saved as `.rds` files and include:
- `G`: Genotype matrix (n x p) with values 0, 1, 2
- `X`: Covariate matrix (n x k) including intercept
- `Y`: Outcome vector (continuous or binary)
- `true_B`: True effect sizes for each variant
- `true_PI`: Causal probabilities (1 for causal, 0 otherwise)
- `causal_idx`: Indices of causal variants
- `maf`: Minor allele frequencies
- `description`: Text description of the dataset

## Available Fixtures

### test_data_simple.rds
- n=100, p=10, k=2
- Continuous outcome
- Independent SNPs (no LD)
- Common variants (MAF: 0.05-0.5)
- 2 causal variants

### test_data_correlated.rds
- n=200, p=20, k=2
- Continuous outcome
- LD structure (r^2 ~ 0.3, AR(1) correlation)
- Common variants
- 4 causal variants

### test_data_binary.rds
- n=500, p=15, k=2
- Binary outcome (balanced cases/controls)
- Independent SNPs
- Common variants
- 3 causal variants

### test_data_continuous.rds
- n=300, p=12, k=3
- Continuous outcome
- Independent SNPs
- Common variants
- Additional covariates (k=3 instead of 2)
- 2 causal variants

### test_data_rare.rds
- n=1000, p=10, k=2
- Binary outcome
- Independent SNPs
- Rare variants (MAF < 0.01)
- 2 causal variants with larger effects

### test_data_sparse.rds
- n=400, p=50, k=2
- Continuous outcome
- Weak LD structure (r^2 ~ 0.2)
- Common variants
- Sparse signal: only 2 causal among 50 variants

## Usage

```r
# Load a fixture
test_data <- readRDS(system.file("validation/fixtures/test_data_simple.rds",
                                  package = "GLOWr"))

# Access components
G <- test_data$G        # Genotypes
X <- test_data$X        # Covariates
Y <- test_data$Y        # Outcome
true_B <- test_data$true_B      # True effects
causal_idx <- test_data$causal_idx  # Causal variants
```

## Regenerating Fixtures

To regenerate all fixtures:

```bash
cd tests/testthat
Rscript generate_fixtures.R
```

This will recreate all `.rds` files with the same random seeds, ensuring reproducibility.
